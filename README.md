#executar com node

1-instalar dependencias do expressjs
cd express-js
npm install

2-instalar dependencias da aplicacao angularjs
cd angular-seed
npm install

3-Subir app angularjs
cd angular-seed
npm start

4-Subir express
cd express-js
node app.js
